## Convention

Chaque fichier sprint contient :
- **Objectif** — pourquoi ce sprint
- **Livré** — liste de ce qui a été construit
- **Fichiers clés** — paths des fichiers créés ou modifiés
- **Décisions** — choix techniques et leur justification
- **Problèmes résolus** — bugs et solutions
- **Blocages / dette** — ce qui reste à finir ou est intentionnellement laissé
